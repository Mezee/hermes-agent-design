# Hermes Agent Design

Hermes Agent Design is a Codex plugin for designing, reviewing, and discovering Hermes agent workflows.

It bundles:
- harness design workflows
- agent-native review workflows
- Hermes design best practices
- Hermes skill discovery helpers

## Included skills

- `$hermes-harness-builder`
- `$hermes-agent-native-reviewer`
- `$hermes-agent-design-best-practices`
- `$hermes-skill-discovery`
